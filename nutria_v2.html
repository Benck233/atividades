API NutriIA
Baseada no Guia MS 2014

Esta aplicação usa a API Anthropic (Claude) com um system prompt especializado baseado nas diretrizes do Guia Alimentar para a População Brasileira — Ministério da Saúde, 2ª edição (2014).

O sistema de prompt incorpora: Classificação NOVA, os 10 Passos, as categorias de alimentos (in natura, minimamente processados, processados, ultraprocessados) e as recomendações de refeições brasileiras tradicionais.

📡 Endpoint Base
POST https://api.anthropic.com/v1/messages
// Anthropic Claude API — claude-opus-4-5
🧠 System Prompt NutriIA (trecho)
// SISTEMA: Nutricionista IA — Guia Alimentar Brasileiro
Você é um nutricionista especializado nas diretrizes do
Guia Alimentar para a População Brasileira (MS, 2014).

// CLASSIFICAÇÃO NOVA:
NOVA 1 — In natura/minimamente processados (PRIORIDADE MÁXIMA)
Frutas, verduras, legumes, grãos, carnes, ovos, leite

NOVA 2 — Ingredientes culinários (use com moderação)
Óleos, gorduras, açúcar, sal, farinha

NOVA 3 — Processados (use com moderação)
Conservas, queijos, pães, embutidos

NOVA 4 — Ultraprocessados (EVITAR)
Refrigerantes, biscoitos recheados, embutidos industriais

// PRINCÍPIOS DO GUIA:
1. Base da alimentação: alimentos in natura e refeições
tradicionais brasileiras (arroz, feijão, hortaliças)
2. Óleos, gorduras, sal e açúcar: usar em pequenas
quantidades para temperar e cozinhar
3. Preferir alimentos produzidos localmente e sazonais
4. Respeitar a cultura alimentar regional brasileira
5. Comer com regularidade e atenção, em ambiente
tranquilo, preferencialmente em companhia
💻 Exemplo de Chamada
// JavaScript — Integração NutriIA API
const response = await fetch('https://api.anthropic.com/v1/messages', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': API_KEY,
    'anthropic-version': '2023-06-01'
  },
  body: JSON.stringify({
    model: 'claude-opus-4-5',
    max_tokens: 1500,
    system: NUTRIA_SYSTEM_PROMPT, // ver abaixo
    messages: [{
      role: 'user',
      content: buildNutriaPrompt({
        ingredients: 'arroz, feijão, frango',
        goal: 'Emagrecer',
        nova: 12, // NOVA 1+2
        time: 30,
        meal: 'almoço',
        savedRecipes: savedList // anti-repetição
      })
    }]
  })
});

// Resposta: JSON estruturado com title, emoji,
// time, calories, protein, carbs, fat, ingredients,
// steps, novaScore, novaAlert, tip, guiaPasso
📊 Campos da Resposta
title Nome criativo da receita
emoji Emoji representativo
novaScore Classificação NOVA (1-4)
novaAlert Alerta nutricional baseado no Guia
calories / protein / carbs / fat Macro-nutrientes
ingredients[] Lista de ingredientes
steps[] Modo de preparo
tip Dica nutricional do guia
guiaPasso Passo do Guia Alimentar referenciado
sustainability Nota de sustentabilidade (1-5)
